# React Native 路由导航使用文档

## 1. 项目导航架构概览

本项目采用 **分层导航架构**，主要包含以下层级：

```
AppNavigator (根导航)
├── AuthNavigator (认证导航)
│   ├── Login
│   ├── Register  
│   └── ForgotPassword
└── TabNavigator (主标签导航)
    ├── HomeStackNavigator
    │   └── Home
    ├── ToolsStackNavigator
    │   └── Tools
    ├── DiscoverStackNavigator
    │   └── Discover
    └── MineStackNavigator
        ├── Mine
        ├── Profile
        └── Settings
```

## 4. 导航器配置

### 4.1 根导航器 (`AppNavigator.tsx`)

```typescript
import React from 'react';
import { NavigationContainer } from '@react-navigation/native';
import { createNativeStackNavigator } from '@react-navigation/native-stack';
import { useSelector } from 'react-redux';

const Stack = createNativeStackNavigator<RootStackParamList>();

export const AppNavigator = () => {
  const isAuthenticated = useSelector((state: RootState) => state.auth.isAuthenticated);

  return (
    <NavigationContainer theme={Theme}>
      <Stack.Navigator screenOptions={{ headerShown: false }}>
        {isAuthenticated ? (
          <Stack.Screen name="Main" component={TabNavigator} />
        ) : (
          <Stack.Screen name="Auth" component={AuthNavigator} />
        )}
      </Stack.Navigator>
    </NavigationContainer>
  );
};
```

### 4.2 标签导航器 (`TabNavigator.tsx`)

```typescript
import { createBottomTabNavigator } from '@react-navigation/bottom-tabs';
import { ROUTES, STACKS } from './core/routers';

const Tab = createBottomTabNavigator<TabParamList>();

export const TabNavigator = () => {
  return (
    <Tab.Navigator screenOptions={({ route, navigation }) => ({
      headerShown: false,
      tabBarStyle: {
        display: !shouldHideTab(navigation) ? 'flex' : 'none'
      },
    })}>
      <Tab.Screen
        name={ROUTES.TabHome}
        component={STACKS.Home}
        options={{
          tabBarLabel: I18n.t('home'),
          tabBarIcon: ({ color, size }) => (
            <Icon name="home-outline" color={color} size={size} />
          ),
        }}
      />
      {/* 其他标签页... */}
    </Tab.Navigator>
  );
};
```

### 4.3 堆栈导航器示例 (`HomeStackNavigator.tsx`)

```typescript
import { createNativeStackNavigator } from '@react-navigation/native-stack';
import { ROUTES, SCREENS } from '@/navigation/core/routers';

      component={SCREENS[ROUTES.Home]}
      options={{ title: I18n.t('home') }}
    />
    {/* 可添加子页面 */}
  </Stack.Navigator>
```

## 5. 页面跳转使用

### 5.1 基本跳转

```typescript
import { useNavigation } from '@react-navigation/native';
import { ROUTES } from '@/navigation/core/routers';

const MyComponent = () => {
  const navigation = useNavigation();

  // 跳转到设置页面
  const goToSettings = () => {
    navigation.navigate('Mine', { screen: ROUTES.Settings });
  };

  // 跳转到个人资料页面（带参数）
  const goToProfile = (userId: string) => {
    navigation.navigate('Mine', { 
      screen: ROUTES.Profile, 
      params: { userId } 
    });
  };

  return (
    <TouchableOpacity onPress={goToSettings}>
      <Text>去设置</Text>
    </TouchableOpacity>
  );
};
```

### 5.2 带参数跳转

```typescript
// 在类型定义中声明参数
export type MineStackParamList = {

  Profile: { userId: string };

// 在页面中接收参数
import { useRoute } from '@react-navigation/native';

  const route = useRoute();
  const { userId } = route.params as { userId: string };

  return (
    <View>
      <Text>用户ID: {userId}</Text>
    </View>
  );
};
```

### 5.3 返回操作

```typescript
const MyComponent = () => {
```

## 7. 最佳实践

### 7.1 路由命名规范

- **Tab 路由**：使用 `Tab` 前缀，如 `TabHome`、`TabTools`
- **页面路由**：使用语义化名称，如 `Home`、`Profile`、`Settings`
- **避免重复**：确保同一导航树下的路由名称唯一

### 7.2 类型安全

```typescript
// ✅ 推荐：使用类型化的导航
const navigation = useNavigation<NavigationProp<RootStackParamList>>();

// ❌ 避免：使用 any 类型
const navigation = useNavigation<any>();
```

### 7.3 参数传递

```typescript
// ✅ 推荐：在类型定义中声明参数
export type ProfileStackParamList = {
  Profile: { userId: string; userName?: string };
};

// ✅ 推荐：使用可选参数
navigation.navigate('Profile', { userId: '123' });
navigation.navigate('Profile', { userId: '123', userName: 'John' });
```

### 7.4 条件导航

```typescript
// 根据认证状态显示不同页面
```

## 8. 常见问题

### Q: 如何隐藏 TabBar？

A: 在 `TabNavigator.tsx` 中使用 `shouldHideTab` 函数：

```typescript
const shouldHideTab = (navigation: any) => {
  const state = navigation.getState();
  return state.routes[state.index].state?.index > 0;
};
```

### Q: 如何添加新页面？

A: 按以下步骤：

1. **添加路由定义**：

```typescript
// core/routers.ts
export const ROUTES = {
  // ... 现有路由
  NewPage: 'NewPage',
} as const;

export const SCREENS = {
  // ... 现有页面
  [ROUTES.NewPage]: NewPageScreen,
};
```

2. **添加类型定义**：

```typescript
// core/types.ts
export type HomeStackParamList = {
  Home: undefined;
  NewPage: { id?: string };
};
```

3. **在堆栈导航器中添加**：

```typescript
// stacks/HomeStackNavigator.tsx
<Stack.Screen
  name={ROUTES.NewPage}
  component={SCREENS[ROUTES.NewPage]}
  options={{ title: '新页面' }}
/>
```

### Q: 如何实现深度链接？

A: 在 `AppNavigator.tsx` 中配置：

```typescript
const linking = {
  prefixes: ['myapp://', 'https://myapp.com'],
  config: {
    screens: {
      Main: {
        screens: {
          TabHome: 'home',
          TabTools: 'tools',
        },
      },
      Auth: 'auth',
    },
  },
};

<NavigationContainer linking={linking}>
  {/* ... */}
</NavigationContainer>
```
